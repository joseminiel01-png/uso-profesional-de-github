# Proyecto fin pag 29

En esta parte del repositorio se desarrolla el proyecto final de la pagina 29 de la guia 3.

El proyecto que definimos fue una pagina web simple para ver la temperatura en Santo Domingo.

El lider del proyecto trabaja en la rama proyecto-fin-pag29.

El frontend trabaja en la rama proyecto-fin-pag29/frontend.

El backend trabaja en la rama proyecto-fin-pag29/backend.
